# Protection flags reference

## Overview

Typewriter exposes protection flags through `RegionFlagKey` and `RegionFlagRegistry`,
which wrap WorldGuard's built-in registry and the ExtraFlags Plus additions. The
registry stores value metadata and attaches default handlers that enforce boolean
flags via the runtime evaluation engine, while inheritance continues to be
resolved through WorldGuard's `FlagValueCalculator` across the parent/child region
hierarchy.【F:extensions/ProtectionExtension/src/main/kotlin/com/typewritermc/protection/flags/RegionFlags.kt†L12-L94】【F:extensions/ProtectionExtension/src/main/kotlin/com/typewritermc/protection/flags/FlagRegistry.kt†L29-L83】【F:extensions/ProtectionExtension/Docs/BTC-Protection/worldguard-core/src/main/java/com/sk89q/worldguard/protection/FlagValueCalculator.java†L49-L178】
